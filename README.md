VOTING SYSTEM APPLICATION

Description

A menu-driven Python application that simulates a real-world election process.
It allows candidate registration, voter authentication, vote casting, and automated result computation.

## Features
- Candidate management (add/view candidates)
- Voter registration with unique ID and password
- Secure login-based voting
- One-person-one-vote enforcement
- Automatic result calculation with tie-handling logic

## Technologies Used
- Python (Core Python)
- VS Code (IDE)
- Git & GitHub (version control)

## How to Run
1. Clone the repository  
2. Open the folder in VS Code  
3. Run the file using:
   python voting_system.py
